\# To-Do List App



A simple To-Do List application built using HTML, CSS, and JavaScript.



\## Features

\- Add new tasks

\- Edit tasks

\- Delete tasks

\- Mark tasks as completed

\- Save tasks using Local Storage



\## Technologies Used

\- HTML

\- CSS

\- JavaScript
